Ce mimi quiz est un projet proposé suite a l'apprentissage des langages php et javascript.
Dans ce projet vous serez en mesure de juger le potentiel de tout un chacun.

les acquis dans ce projet sont les validations coté serveur et coté client.
Bonne maitrise des bases de javascript et plus important en php.

Nous avons developpes une partie admin et une partie joueur dans ce quiz.
pour acceder a ces partie veuillez utilisez ces coordonnees suivant:
partie admin : login= admin et mdp=admin
partie joueur :login=joueur et mdp=joueur
sinon vous serez agacé par nos message d'erreur car etant bien validé!

pour tester notre travaille veuillez cliquer sur le lien suivant:
http://abdquizz.alwaysdata.net/
pour visualisez les codes sources voici le lien github:
https://github.com/abdou-karim/Abdou-karim-miniprojet-sa
Dans ce pure projet nous avons utilise comme technologie les langages suivants:
Html , Css , Javascript et Php

Ce projet est un jeux de quiz pas comme les autres devenez admin et vous aurez la capacité de creer vos propres questions , de donnez les scores et de fixer le nombre de question par jeux.
l'administrateur a la capacité d'ajouter d'autre admin.
Un plaisir de jouer est reserve aux joueurs avec un designe bien travaillé.
